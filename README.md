## Stack Tecnológico

**Cliente:** Ionic, Angular, PrimeNG, Swiper

**Servidor:** Symfony, API Platform


## Iniciar Ionic App

Inicio Normal - Entorno de Pruebas Estandar

```bash
  ionic serve
```
Inicio Móvil - Entorno Android / Iphone

```bash
  ionic serve --lab
```

## Iniciar Symfony App

Inicio Normal - Entorno de Pruebas Estandar

```bash
  symfony server:start
```
https://localhost:8000/api -> API Platform
